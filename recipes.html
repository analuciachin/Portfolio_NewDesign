Recipes

Recipes is a web application that helps the user manage their list of recipes, search for recipes based on meal type and select their favourite recipes. The API used on this project comes from https://dummyjson.com/

Project Purpose and Goal

Recipes is a project I created to improve my understanding of the foundation and core concepts of React. In this project, I used the Dummy JSON API to get a dataset of sample recipe data and focused only on the front-end portion of the application.

>

Recipes - Docs: https://dummyjson.com/docs/recipes

Homepage - the page fetches all the recipes from the API. The user can delete recipes, toogle their favourite recipes and see a recipe's details.

Recipe details - the details page contains more information about a specific recipe like prep/cook time, number of servings, level of difficulty, type of cuisine, calories per serving, meal types, ingredients and instructions on how to prepare it.

Favourite - the tab shows at the top the number of favourite recipes selected by the user. Once a recipe is toggled as favourite (yellow star), it automatically displays in the Favourite tab. If he user toggles again it's removed from the favourite list.

Search Recipes - the user can search recipe by meal type and check the recipe's details

Problems and Thought Process

Like most projects, I ran into a few bumps along the way, one of them was the code organization and project architecture. I had drafted a simple project and as it came to life I wanted to add more features so I had to break it down into smaller components.

Lessons Learned

Working on My Movies project gave me the chance to learn, review and practice some Javascript concepts that I was not very confortable with such as promises, ES6 syntax and object manipulation. It also gave me the opportunity to get more familiar with API documentations, using POSTMAN to test and make calls to the API, practicing React's setState callbacks and working with Bootstrap for responsive design.

Future Improvements

The next steps for this project would be having a backend where I can store all the data, so the user does not lose track of all his activities after the browser refreshes. Also, I would like refactor the code to make the components as reusable as possible and add React Hooks to it.
